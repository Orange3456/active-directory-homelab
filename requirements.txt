# ============================================================
# Active Directory Enterprise Homelab - Requirements
# ============================================================
# This repository documents a hands-on homelab build.
# No code dependencies are required to use this repo.
# The requirements below list the SOFTWARE and HARDWARE
# needed to reproduce the lab environment.
# ============================================================

# ------------------------------------------------------------
# HARDWARE REQUIREMENTS
# ------------------------------------------------------------
# Minimum:
#   CPU    : Intel i5 or AMD Ryzen 5 (4 cores, virtualisation support)
#   RAM    : 16 GB (lab runs 2-3 VMs simultaneously, never all 5 at once)
#   Disk   : 100 GB free space (all VMs use thin provisioning)
#   OS     : Windows 10/11 host (VMware Workstation requirement)
#
# Tested on:
#   Dell Latitude - Intel i5 - 16 GB RAM - 512 GB SSD

# ------------------------------------------------------------
# SOFTWARE REQUIREMENTS (all free for personal/lab use)
# ------------------------------------------------------------

VMware-Workstation-Pro>=17.0.0

# Free for personal use under current VMware licensing
